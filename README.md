# To-Do List React

Este é um projeto de lista de tarefas desenvolvido com React e TypeScript. Ele permite adicionar, alternar o estado de conclusão e remover tarefas.

## Funcionalidades

- **Adicionar Tarefa**: Insira uma nova tarefa com um texto e estado inicial.
- **Alternar Conclusão**: Marque ou desmarque uma tarefa como concluída.
- **Remover Tarefa**: Exclua uma tarefa da lista.

## Estrutura do Projeto

A estrutura principal do projeto é a seguinte:

```
public/          # Arquivos estáticos
src/
  app/           # Componentes principais da aplicação
  components/    # Componentes reutilizáveis
  contexts/      # Context API para gerenciamento de estado
  reducers/      # Redutores para manipulação de estado
  types/         # Definições de tipos TypeScript
```

## Tecnologias Utilizadas

- **React**: Biblioteca para construção de interfaces de usuário.
- **TypeScript**: Superset do JavaScript para tipagem estática.
- **Next.js**: Framework React para renderização do lado do servidor e geração de sites estáticos.

## Como Executar o Projeto

1. Clone o repositório:
   ```bash
   git clone <URL_DO_REPOSITORIO>
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

4. Acesse o projeto no navegador:
   ```
   http://localhost:3000
   ```
